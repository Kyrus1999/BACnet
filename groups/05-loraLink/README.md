# README (in progress)

- Pycom LoPy4 (https://pycom.io/product/lopy4/) 
- Heltec WiFi LoRa 32 (V2)

Software:
- Pycom LoPy4 with Micropython
- Heltec with Arduino Development Environment

- alternative communication net for small packages
